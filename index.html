Apprendre Git&GitHub

Liste des Commandes Git

  • git init : Initialisation d'un projet git
  • git branch -M main: Renomer la branche principale par "main"
  • git status: Voir tous les changements apportées au projet
  • git add. || git add * || git add --all || git add "nom fichier": Ajouter un ou plusieurs fichier dans l'index
  • git commit -m "nom du commit": Envoyer tous les fichiers indexés vers le depots git
  • git log | git log --all: Permet de voir tous les logs ( notamment les commits )
  • git log --oneline: Permet d'afficher les logs sur chaque ligne

  • git diff | git diff "nom fichier" : Permet de voir tous les modifications faites dans le projet ou dans un fichier specifique
  • git diff id_commit_1 id_commit_2 | git diff id_commit_1 id_commit_2 "nom fichier specifique": Permet de comparer les modifications faites dans la commit 1 par rapport à la commit 2 et on peut preciser un fichier specifiquer pour faire la comparaison
  • git checkout id_commit_1: Permet de basculer vers un different commit selon son id
  • git checkout "nom branche": Permet de se mettre dans la dernière commit
